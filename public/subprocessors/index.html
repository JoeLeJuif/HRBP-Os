Sous-traitants

Dernière mise à jour : 13 mai 2026

HRBP OS s'appuie sur un nombre restreint de sous-traitants techniques pour fournir le Service. Cette page liste les sous-traitants en production et leur rôle. Les modifications sont communiquées via la Politique de confidentialité.

Supabase

Rôle : hébergement de la base de données PostgreSQL, gestion de l'authentification (magic link), application des politiques de sécurité au niveau des lignes (Row Level Security).

Données traitées : données de compte, contenu RH applicatif, journaux d'audit.

Localisation : infrastructure cloud, régions selon la configuration du projet (peut inclure hors Canada).

Site : supabase.com

Vercel

Rôle : hébergement et livraison de l'application web (front-end et fonctions API), réseau de distribution de contenu (CDN), journaux d'accès opérationnels.

Données traitées : métadonnées de requêtes HTTP (IP, agent utilisateur), trafic vers les fonctions API.

Localisation : infrastructure cloud, principalement aux États-Unis et en Europe.

Site : vercel.com

Anthropic

Rôle : traitement par modèle de langage (Claude) pour les fonctions assistées par IA — génération de briefs, analyse de signaux, suggestions, structuration de cas.

Données traitées : uniquement le contenu transmis par l'utilisateur dans le cadre d'une requête IA (cas, notes, contexte de rencontre, etc.).

Localisation : infrastructure cloud aux États-Unis.

Site : anthropic.com

Transferts hors Québec/Canada

En raison de la localisation des sous-traitants ci-dessus, certaines données peuvent être traitées hors du Québec et du Canada, notamment aux États-Unis. Voir la Politique de confidentialité pour plus de détails.

Mises à jour

Tout ajout d'un nouveau sous-traitant traitant des données client sera reflété sur cette page. Les clients ayant un engagement contractuel formel seront notifiés selon les modalités prévues à leur entente.
